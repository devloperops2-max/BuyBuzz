# **App Name**: Lumina

## Core Features:

- User Authentication: Secure sign-up/sign-in with interest selection.
- Admin Dashboard: Admin access for product and order management.
- Product Management: Add, edit, and delete products with details including name, description, price, category and optional image.
- AI Assistant: Nova: AI-powered chatbot for personalized product recommendations and conversational assistance, leveraging user interests, browsing history, search and purchase behavior. Uses an AI API as a tool.
- Smart Target Marketing: AI dynamically updates recommendations based on user interactions and learns from searches, viewed products, and purchases.
- Database Storage: Data stored securely using MongoDB Atlas or Firebase; admin preloaded.
- Responsive UI/UX Design: Minimalistic design with clean typography and floating chat icon for Nova.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calming and trustworthy atmosphere.
- Background color: Very light gray (#F5F5F5) to maintain a minimalistic feel and allow products to stand out.
- Accent color: Pale violet red (#D87093) for highlighting interactive elements and calls to action.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral look.
- Simple, line-based icons for categories and UI elements.
- Clean, grid-based layout with ample white space.
- Subtle transitions and animations for a smooth user experience.
